# course-pizzashop-api-elysia-bun

> API em desenvolvimento - Curso **API com Bun** da Rocketseat, utilizando **Bun**, **Elysia** e **Drizzle ORM**.

---

## Sobre o Projeto

Este projeto implementa uma **API para um painel de gerenciamento de pedidos** (semelhante a sistemas de restaurantes e pizzarias) construída com o runtime **Bun**.  
O objetivo é aprender na prática a estruturar APIs performáticas, seguras e escaláveis com **Elysia** e **Drizzle ORM**, explorando desde o setup inicial até autenticação e rotas complexas.

---

## Stack Técnica

| Tecnologia | Função |
|-------------|---------|
| **Bun** | Runtime rápido e moderno |
| **Elysia** | Framework para Bun |
| **Drizzle ORM** | ORM para PostgreSQL |
| **Docker Compose** | Container e orquestração do banco PostgreSQL |
| **PostgreSQL** | Banco de dados |
| **Nodemailer** | Envio de e-mails |
| **ESLint** | Padronização e qualidade do código |
| **JWT (JSON Web Token)** | Autenticação e autorização seguras |

## Setup do Ambiente

```bash

```
